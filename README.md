### Planteamiento

El Sistema de Pedidos PoliBurguer es una aplicación web diseñada para soportar los procesos operativos del restaurante de comida rápida “PoliBurguer”. Su finalidad es optimizar la atención al cliente y la gestión interna de la cocina mediante un canal digital de recepción y seguimiento de pedidos.
El sistema busca mejorar la eficiencia operativa, reducir tiempos de espera y minimizar errores en la toma de pedidos, modernizando el proceso típico de atención en un restaurante de comida rápida.
